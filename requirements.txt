# 🚀 Setup Completo Python + Flask + Git (Windows)

## 🐍 1. Criar e configurar ambiente virtual

```bash
# Criar ambiente virtual
python -m venv venv

# Ativar ambiente (Windows)
venv\Scripts\activate

# Atualizar pip (opcional)
python -m pip install --upgrade pip

# Instalar Flask
pip install flask

# Criar arquivo de dependências (manual)
echo flask > requirements.txt
```

---

## ▶️ 2. Criar aplicação Flask

Crie um arquivo chamado `app.py`:

```python
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Funcionando 🚀"

if __name__ == "__main__":
    app.run(debug=True)
```

Rodar o projeto:

```bash
python app.py
```

Acesse no navegador:

```
http://127.0.0.1:5000/
```

# Primeiro commit

# Enviar para o GitHub

```

## 4. Configurar Git (Máquina Compartilhada)

```bash
# Iniciar repositório
git init

# Configurar usuário SOMENTE neste projeto (sem --global)
git config user.name "Seu Nome"
git config user.email "seu@email.com"

# Conectar ao repositório remoto
git remote add origin https://github.com/seu-user/seu-repo.git

# Renomear branch para main (padrão atual do GitHub)
git branch -M main

# Adicionar todos os arquivos ao stage
git add .

# Criar o primeiro commit
git commit -m "primeiro commit"

# Enviar para o GitHub e criar vínculo (upstream)
git push -u origin main
```

> A partir do segundo push, basta usar:
> ```bash
> git push
> ```
