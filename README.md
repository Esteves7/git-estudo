# 💰 Tip Calculator - Modern Dark Mode

Uma calculadora de gorjetas elegante e funcional, projetada com uma interface **Modo Escuro Moderno** de alto contraste e estética tecnológica.

---

## 🌟 Destaques do Design

Este projeto foi atualizado para oferecer uma experiência visual superior, focada em conforto ocular e clareza de informações:

*   **Fundo Profundo**: Utiliza tons de cinza escuro (`#121212`) para reduzir a fadiga visual.
*   **Texto de Alto Contraste**: Todas as fontes principais são **puramente brancas** (`#ffffff`), garantindo leitura perfeita.
*   **Acentos Vibrantes**: Cores neon (azul, verde e vermelho) são usadas estrategicamente para indicar estados e ações.
*   **Interface Responsiva**: Adaptada para dispositivos móveis com ajustes automáticos de tipografia e espaçamento.

---

## 🚀 Funcionalidades

- [x] Cálculo instantâneo de gorjetas.
- [x] Divisão de conta entre várias pessoas.
- [x] Classificação visual de resultados (Sucesso, Info, Perigo).
- [x] Interface totalmente adaptada para o Modo Escuro.
- [x] Campos de entrada otimizados com foco visual.
